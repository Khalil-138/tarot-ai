# 🔮 Tarot AI

Projeto de portfólio desenvolvido para praticar programação web e construção de experiências conversacionais.

O projeto utiliza HTML, CSS, JavaScript, Node.js e Git para criar um tarô visual interativo com leitura simbólica e experiência conversacional.

## ✨ Funcionalidades

- Retirada aleatória de 3 Arcanos Maiores
- Cartas visuais personalizadas
- Interface conversacional
- Animações e revelação das cartas
- Estrutura preparada para integração com IA

## 🛠 Tecnologias

- HTML5
- CSS3
- JavaScript
- Node.js
- Express
- Git / GitHub

## 🎯 Objetivo

Este projeto foi criado como exercício de aprendizado e portfólio, explorando desenvolvimento web, design de interação e interfaces conversacionais.

## 🚧 Próximos passos

- Integração com IA real
- Leituras dinâmicas
- Melhorias visuais
- Deploy online
